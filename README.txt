Это мой первый pet-проект, который я пишу для анализа синтетических логов системы обнаружения вторжений.
Для этого я использую датасет "Intrusion detection log analysis" с сайта Kaggle.

Я хочу исследовать данный лог-файл, классифицировать данные в нём и создать модель, классифицирующую тип вредоносной активности.

Почему я выбрал данную тему для моего первого проекта?
Я аудитор информационной безопасности, но попробовать себя в Data Science. Поэтому я решил объединить эти две сферы. На самом деле, я никогда не работал с анализом логов, потому что это не моё направление ИБ, но я думаю, что это будет интересный опыт.

Ссылка на датасет: https://www.kaggle.com/datasets/developerghost/intrusion-detection-logs-normal-bot-scan

This is my first pet-project, which I write for analyzing synthetic logs of an Intrusion Detection System.
For this purpose I use the "Intrusion detection log analysis" dataset from Kaggle.

So I want to analyze this log file, classify an information within and create a model to classify a type of malicious activity.

Why did I chose this topic for my first project?
I am an information security auditor, who want to become a data scientist. So I decided to join these two sphere. I'm actually didn't work with logs analysis before, because it's not my direction of information security, but I think it is interesting experience.
